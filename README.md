This project is a relational database system that simulates a personal finance management platform for users across the UAE. The database stores user profiles, bank accounts, financial transactions, budgets, and spending categories, enabling users to track income, expenses, savings, and budgeting performance. Using PostgreSQL, the system supports analytical queries and reporting views that provide insights into spending behaviour, budget compliance, user activity, and financial trends. The database is designed to demonstrate how transactional financial data can be transformed into meaningful business intelligence to support both product improvements and strategic decision-making.
